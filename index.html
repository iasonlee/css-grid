I'm working on the perfect grid for a blog site that uses a single page template. The template is just about the placement of items to make sure that regardless of titles or lengths of different text items, the overall look and feel is almost identical. I'll be making changes and making the CSS smaller and smaller until it is optimized. I'm going to be using the grid myself on my own blog once I finish + polish things up.

The grid is set up and ready for things like code snippets: 

hi = ["hi", "look", "at", "this", "code-block"]
for i in hi:
  print(i)
print('hello and welcome to my new site where code can be horizontally scrolled')
print('also vertically scrollable')
bye = ["ok", "goodbye", "now"]
for j in bye:
  print(j)

and even inline TeX writing with Math-Jax. $\forall x\in\mathbb{R},\exists{y}\in\mathbb{R}:x\le y$ completely inline and allows you to continue writing.

Features

  • Modern theme
  • Scrollable code blocks with highlighting
  • Inline TeX
  • Automatic dark mode / light mode
  • Grid layout that works seamlessly with many types of content and works well on mobile
